# ❄️ Efecto Cristal (Glassmorphism)

Proyecto de demostración de **Glassmorphism / efecto cristal** en CSS.  

Actualmente contiene texto de ejemplo, pero está pensado para mostrar cómo crear paneles o elementos con **trasparencia, blur y bordes difuminados**, logrando un efecto de vidrio moderno.

---

## 🧩 Idea del proyecto

- Crear elementos que simulen **cristal o vidrio**.
- Aplicar **`backdrop-filter: blur()`** y **transparencias**.
- Añadir **bordes suaves y sombreados** para realismo.
- Ideal para **cards, ventanas modales o secciones UI**.
